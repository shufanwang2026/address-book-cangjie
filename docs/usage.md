# 使用说明

## 1. 环境准备

请先安装仓颉工具链，并确保命令行可以识别 `cjpm`。

在 Windows 中可以打开 PowerShell，输入：

```bash
cjpm --version
```

如果能看到版本号，说明环境已经配置好。

## 2. 运行项目

进入项目目录：

```bash
cd E:\project
```

运行程序：

```bash
cjpm run
```

## 3. 手动测试建议

可以按照下面步骤检查功能是否正常：

1. 添加一个联系人，例如姓名 `张三`
2. 查看所有联系人，确认能看到 `张三`
3. 按姓名查找 `张三`
4. 修改 `张三` 的电话
5. 删除 `张三`
6. 再次查看所有联系人，确认联系人已经删除

## 4. 常见问题

如果控制台中文显示乱码，可以先执行：

```bat
chcp 65001
```

如果提示找不到 `cjpm`，说明仓颉工具链没有安装好，或者没有配置环境变量。
