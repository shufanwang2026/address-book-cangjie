# 通讯录系统设计说明

## 1. 设计目标

本项目做一个简单的控制台通讯录系统，重点练习变量、数组、函数、类、循环和条件判断。

## 2. 数据设计

联系人使用 `Contact` 类表示，包含四个字段：

- `name`：姓名
- `phone`：电话
- `email`：邮箱
- `address`：地址

通讯录使用 `Array<Contact>` 保存联系人。为了降低难度，项目设置最多保存 100 个联系人，并用 `count` 记录当前实际人数。

## 3. 功能设计

- 录入：把新联系人放到数组的 `count` 位置，然后 `count + 1`
- 删除：先查找联系人，再把后面的联系人向前移动一格
- 显示：从下标 `0` 遍历到 `count - 1`
- 查找：按姓名逐个比较，找到后返回数组下标
- 修改：先查找联系人，再用新输入的信息覆盖旧联系人
- 保存：把内存中的联系人写入 `contacts.txt`
- 加载：读取 `contacts.txt`，把每一行转换成联系人

## 4. 为什么这样写

这个版本没有使用数据库、文件保存和复杂集合，主要是为了符合大一水平。固定数组的写法可以清楚展示“下标”“遍历”“移动元素”等基础概念，也便于老师检查代码逻辑。

## 5. 文件格式

保存文件名为 `contacts.txt`。每一行表示一个联系人，字段顺序如下：

```text
姓名    电话    邮箱    地址
```

字段之间使用制表符分隔。这样既方便程序读取，也方便打开文件检查内容。
