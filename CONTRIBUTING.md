# 协作说明

这个项目适合课堂练习和同学之间互相学习。为了方便大家阅读和修改，请尽量遵守下面的规则。

## 代码规则

- 保持代码简单，优先使用大一阶段容易理解的写法
- 新增函数时写清楚函数用途
- 不要一次修改太多无关内容
- 中文提示语要清楚，方便运行时理解

## 提交建议

提交信息可以使用下面格式：

```text
feat: 增加文件保存功能
fix: 修复删除联系人后的显示问题
docs: 更新运行说明
```

## 推荐分工

- 一位同学负责菜单和输入输出
- 一位同学负责联系人数组的增删改查
- 一位同学负责文档和测试记录
- 一位同学负责后续扩展，例如文件保存
